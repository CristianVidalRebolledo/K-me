## 2. El Concepto de "Entorno Cerrado de Servicios"

### Qué significa

```
ANTES (cliente solo con distribuidora):
Pyme
  │
  └─→ Enel/CGE
       ├─ Venden energía
       ├─ Miden consumo
       ├─ Facturan
       └─ ❌ No optimizan (porque si baja demanda, pierden ingresos)

DESPUÉS (cliente con tú + distribuidora):
Pyme
  ├─→ TÚ (ESCO)
  │   ├─ Diagnosticas (telemetría)
  │   ├─ Propones soluciones (BESS, thermal, demand response)
  │   ├─ Instalas y monitoreas
  │   ├─ Optimizas automáticamente
  │   ├─ Generates data insights
  │   └─ ✅ INCENTIVADO: Ganas % del ahorro
  │
  └─→ Enel/CGE
       ├─ Siguen vendiendo energía (menos cantidad, OK)
       ├─ Siguen midiendo
       ├─ Facturan menos (cliente consume menos en punta)
       └─ Se benefician: menos presión en red en punta
```

### El "Entorno Cerrado"

```
Significa: El cliente NO busca múltiples proveedores
├─ Problema energético → Llama a TÍ (acceso exclusivo)
├─ Propones solución integral → No necesita otro proveedor
├─ Monitoreo continuo → "Pegado" al cliente
├─ Ahorros verificados → Cliente ve valor permanente
├─ Recomendaciones automáticas → Sigue tus consejos
└─ Alto switching cost: Cambiar de ESCO es costoso/complejo

RESULTADO:
├─ Retención 90%+ (vs SaaS típico 80-85%)
├─ Expansion revenue: Nuevas soluciones cuando se identifiquen
├─ LTV muy alto: Cliente está 5-7 años contigo
└─ Defensible: Competencia difícil porque requiere:
   ├─ Capital (BESS es caro)
   ├─ Expertise técnico (M&V, regulación)
   └─ Red de instaladores (electricistas socios)
```

## 3. Ciclo de Servicio Completo (6-36 meses)

### FASE 0: Prospección (Semana 1-4)

```
Objetivo: Identificar cliente con "high pain"

Filtro inicial (vía electricista socio):
├─ ¿Cliente está pagando recargo por factor de potencia?
├─ ¿Tiene demanda máxima en horario punta (18-23h)?
├─ ¿Tiene carga térmica (máquinas, neveras, hornos)?
├─ ¿Factura eléctrica > $500k CLP/mes?
└─ ¿Dispuesto a invertir en eficiencia?

Si SÍ a 2+ preguntas → Es candidato
```

### FASE 1: Diagnosis Energética (Semana 1-6)

```
Actividad: "Energetic Audit" usando telemetría + análisis

Entregables:
├─ Instalación temporal de medidor (Eastron SDM630)
├─ 30 días de datos: V, I, P, Q, cosϕ, THD, demanda máxima
├─ Análisis de factura histórica (últimos 12 meses)
├─ Simulación de ahorros por escenario:
│  ├─ Escenario A: Solo factor de potencia → $150k CLP/mes
│  ├─ Escenario B: Factor + peak shaving (BESS) → $220k CLP/mes
│  ├─ Escenario C: Factor + BESS + thermal → $280k CLP/mes
│  └─ Escenario D: Factor + BESS + thermal + solar → $350k CLP/mes
│
├─ ROI por escenario (payback, NPV 5 años)
├─ Opciones de financiamiento:
│  ├─ FFEE (subsidio 50%, cliente paga 50%)
│  ├─ BancoEstado línea energía
│  ├─ Modelo EPC (tú financias, cliente paga % ahorro)
│  └─ Inversor privado (third party)
│
└─ Reporte PDF profesional (para gerente, contador, dueño)

Costo para cliente: $0-500k CLP (incluido en proyecto o mínimo fee)
Tus horas: 40-60 horas
Tu revenue: $500k-1.5M CLP en comisión o fee
```

```
│  └─ Margen para ti
│
├─ Propuesta de contrato EPC (si aplica):
│  ├─ "Te finanzamos, cobras % del ahorro verificado"
│  ├─ M&V: Cómo medimos el ahorro (ISO 50006)
│  ├─ Comisión: 20-30% del ahorro durante 5 años
│  └─ Terminas cuando se pague inversión o 7 años

├─ Gestión de financiamiento:
│  ├─ Postulación FFEE (si es elegible)

Tu revenue:
```

```
Objetivo: Garantizar ahorros, optimizar permanentemente, generar CEE

Actividades mensuales:

Tu revenue (recurrente):
├─ SaaS: $40k-80k CLP/mes × 60 meses = $2.4M-4.8M CLP
```

## 4. Modelo EPC vs SaaS vs Híbrido

### Opción C: Híbrido (Óptimo)

```
Cliente paga:
├─ SaaS base: $30k CLP/mes (monitoreo + control)
├─ Comisión por ahorro: 15% adicional del ahorro
└─ TOTAL: $30k + (ahorro × 15%)

VENTAJA:
├─ Tú tienes ingreso base (predecible)
├─ Cliente tiene incentivo compartido
├─ Flexible si ahorro es menor/mayor
└─ Mejor retención (menos churn por "no ahorro")

EJEMPLO CLIENTE:
├─ Ahorro real: $120k CLP/mes
├─ Pago tuyo: $30k (SaaS) + $18k (15% comisión) = $48k/mes
├─ ROI cliente: -$48k pero -$120k de energía = +$72k net/mes
└─ Cliente feliz, tú feliz

RECOMENDACIÓN: ESTE. Es el balance.
```
